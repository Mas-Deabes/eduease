# EduEase

A web-based educational support platform built with React and Firebase. Designed to increase student engagement in online learning through anonymous discussion, progress tracking and at-risk detection.
Live Demo

https://eduease-self.vercel.app

Tech Stack

React + Vite
Firebase Authentication + Firestore
React Router
Recharts
Vercel

Getting Started

Clone the repository and install dependencies

git clone https://github.com/Mas-Deabes/eduease.git
cd eduease
npm install

Create a .env file with your Firebase credentials

VITE_FIREBASE_API_KEY=your_value
VITE_FIREBASE_AUTH_DOMAIN=your_value
VITE_FIREBASE_PROJECT_ID=your_value
VITE_FIREBASE_STORAGE_BUCKET=your_value
VITE_FIREBASE_MESSAGING_SENDER_ID=your_value
VITE_FIREBASE_APP_ID=your_value

Start the development server

npm run dev
